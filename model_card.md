# Model Card — PawPal+

## Reflection

This project demonstrated that high quality AI applications require a strong foundation of accurate, well organized data. Training a model or building a retrieval system is only as good as the information behind it -- garbage in, garbage out. The knowledge base in PawPal+ made that real. Every piece of advice the AI gave was only as useful as the guidelines that were retrieved. The RAG workflow reinforced how important it is to maintain the right data at the right time, so that the information that reaches the user is relevant and trustworthy rather than generic and doesn't apply to the user. I collaborated with AI by using Claude to help implement the RAG workflow. It was able to create a knwowledge base and a pipeline to utilize the data with an LLM seamlessly. I asked clarifying questions to make sure that users can easily gain insights about their pet. I also used its Ask mode to have questions answered before real implemetations to the code. Keeping a constant check allowed me to make sure that the code maintained accurate.
